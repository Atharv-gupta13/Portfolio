# Dictionary App - Learn Word Meanings Instantly

📚 A feature-rich dictionary application with advanced search capabilities!

## Features
🔍 Instant word search with suggestions
📖 Complete word definitions and meanings
🔊 Pronunciation guides (with audio when available)
📝 Example sentences for context
🌳 Word etymology and origins
⭐ Save favorite words for later
🌙 Dark/Light mode theme
📱 Fully responsive design

## Tech Stack
- HTML5, CSS3, JavaScript
- REST API (Dictionary API integration)
- Local Storage for favorites
- Responsive Grid & Flexbox
- Smooth animations

## Live Demo
[Open Dictionary App](https://dictionarye.netlify.app/)

## How to Use
1. Type any English word in the search box
2. View definitions, pronunciation, and examples
3. Click the speaker icon to hear pronunciation
4. Save words to your favorites list
5. Switch between light and dark modes

## API Integration
- Integrated with free Dictionary API
- Real-time data fetching
- Error handling for invalid words
- Caching for better performance

## Features Explained
- **Word Search**: Auto-complete suggestions
- **Rich Content**: Definitions, examples, etymology
- **Audio Support**: Hear correct pronunciation
- **Favorites**: Save words for revision
- **Theme Toggle**: Dark mode for comfortable reading
